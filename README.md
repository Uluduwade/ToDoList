# ToDoList
The initial app was created using MERN (Mongo DB, Express js, React, Node js)
# Execute
to run the app first you have to change your directory in to server and install all the dependencies using

**npm install**

and after that you have to start the server using
**npm start**

after that you have to change the directory to client and install the dependencies in that as well using

**npm install**

and then you can run 

**npm run dev**

in your terminal there will be a link or simply open your browser and go to
http://localhost:5173/
where you'll get the ToDoList

# What can you do

With this app, you can:
- **Create** new tasks
- **Delete** tasks you no longer need
- **Reorder** tasks by moving them up and down to prioritize your workflow
- **Edit** existing tasks
- **Mark** tasks as complete or ongoing
